# knights

Visualizes three chess-knight problems on a number-spiral grid (from a Numberphile
video ‚Äî see `KNIGHTS.md` for the problem statements and validated outcomes). One
std-only Cargo binary, **zero external dependencies**; SVG and PNG output are
hand-rolled.

> Keep this file thin and pointer-shaped. The *how* and *why* live in each module's
> doc-comment; the facts that must stay true live in the tests. Don't restate either
> here ‚Äî a standalone doc is the first thing to drift.

## Layout

- `src/main.rs` ‚Äî CLI: subcommand dispatch + flag parsing; chooses SVG vs PNG.
- `src/spiral.rs` ‚Äî square-spiral coordinates: `SpiralWalker` (O(1) stepper) and `Spiral` (cached tables built from it).
- `src/knight.rs` ‚Äî Problem 1, the trapped knight (and the 8 knight offsets, shared).
- `src/courteous.rs` ‚Äî Problem 2, courteous knights + cluster grouping.
- `src/redblack.rs` ‚Äî Problem 3, red & black: one byte-per-cell grid that is occupancy and raster at once.
- `src/render.rs` ‚Äî SVG rendering for all three.
- `src/raster.rs` ‚Äî streaming indexed-color PNG encoder + the courteous rasterizer.

Each file's module doc-comment explains its design ‚Äî start there.

## Run / test

- `cargo test` ‚Äî includes the verification oracles below.
- `cargo run -- [trapped|courteous|redblack] [flags]` (default: `trapped`, SVG into `out/`).
- `cargo run --release -- redblack --radius 2000` for large boards.

Flags: `--radius <r>`, `--start <n>`, `--format svg|png` (PNG is board-problems only),
`--canvas <px>` (SVG size, default 1600), `--squaresize <px>` (PNG pixels/square, default 1),
`--variant canonical|rot180|mirror|quad` (redblack only; `rot180`/`mirror` reorient
Red's spiral, `quad` plays four colors ‚Äî all non-canonical experiments, see
`redblack.rs`), `--out <path>`. See `cargo run -- --help`.

## Verification oracles (enforced by tests ‚Äî trust these, not prose)

- Trapped knight traps at square **2084** (center = 1; OEIS A316667) ‚Äî `knight::tests::trapped_at_2084_with_start_1`.
- Courteous = OEIS **A308885** ‚Äî `courteous::tests::matches_oeis_a308885`.
- Red & Black = OEIS **A392177** (Black) / **A392178** (Red) ‚Äî `redblack::tests::matches_oeis_a392177_a392178`.

## Conventions

- A PNG cell is a `u8` palette index; the encoder picks the bit depth from the palette
  size, so adding piece types needs no encoder change.
- Zero external dependencies (std only) is deliberate ‚Äî don't add crates without asking.
- `out/` and `target/` are gitignored. `out/` may hold images generated by hand ‚Äî check
  a file's contents before overwriting it.
